# market_place

still in progress. not finished yet.

this will be a maven multi-module microservices project.

## planned stack

- spring boot
- spring cloud gateway
- circuit breaker
- grpc and kafka for inter-service communication
- kubernetes for orchestration and service discovery
- grafana stack for observability
- angular frontend

## frontend

the frontend will be a slightly modified version of this repo:

[angular-10-tut](https://github.com/Mousum2002/angular-10-tut.git)

## current status

for now i started working on the backend and couple of base service is done.
more services and setup will be added later.
